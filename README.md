# DSA_Prep

A comprehensive collection of **Data Structures and Algorithms** solutions, featuring LeetCode problems, HackerRank challenges, and DOM practice exercises.

## 📋 Overview

This repository contains practice solutions for coding interviews and technical assessments, focusing on:
- **Array & String Problems** - Classic data structure challenges
- **Algorithm Implementations** - Various problem-solving techniques
- **Interview Preparation** - LeetCode and HackerRank problem solutions
- **Web Development** - DOM manipulation and JavaScript practice

## 🛠️ Technologies Used

- **Python** (83.7%) - Primary language for DSA solutions
- **HTML** (16.3%) - Web development practice

## 📁 Repository Structure

### LeetCode Solutions
- `leet#125.py` - Valid Palindrome

- `Two_sum#1.py` - Two Sum (LeetCode #1)

  - String manipulation
  - Array operations
  - Algorithm optimization
  - Problem-solving patterns

### Web Development
- `dom_practice.html` - DOM manipulation practice exercises

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- A code editor (VS Code, PyCharm, etc.)
- Git

### Cloning the Repository

```bash
git clone https://github.com/Ysaibhanu99/DSA_Prep.git
cd DSA_Prep
```

### Running Solutions

To run any Python solution:

```bash
python3 filename.py
```

For example:
```bash
python3 Two_sum#1.py
python3 leet#125.py
```
